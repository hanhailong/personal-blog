## Welcome

This is my personal blog, built with **Next.js** and **Markdown**. I wanted something simple, fast, and easy to maintain.

### Why Markdown?

Markdown lets me write in plain text while still producing rich, formatted content. It's:

- **Portable** — just `.md` files, no database needed
- **Version-controllable** — Git diffs are human-readable
- **Distraction-free** — focus on writing, not formatting

```python
def fibonacci(n: int) -> int:
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(10))  # 55
```
